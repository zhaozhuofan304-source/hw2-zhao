# Report

## What worked
The system successfully generates polite and professional customer support replies using the LLM API.

## What didn’t work
The model may produce generic responses and sometimes lacks specific guidance for edge cases.

## Improvements
- Added rules to avoid hallucination
- Added instructions to request missing information
- Defined human review boundaries for sensitive cases

## Human review boundary
Cases involving fraud, threats, or unclear requests should be escalated to a human agent.
