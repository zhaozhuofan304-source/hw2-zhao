# Week 2: Simple GenAI Workflow

## Business Workflow
I chose a customer support response drafting workflow.

## User
The user is a customer support staff member at a small online retail company.

## Input
The system receives a customer message, such as a complaint, refund request, shipping issue, or product question.

## Output
The system should produce a professional first-draft email reply in English that is polite, clear, and helpful.

## Why This Task Is Valuable
This task is valuable to automate because customer support teams often answer repetitive written requests. A first-draft response can save time, improve consistency, and help staff handle more cases efficiently. However, some sensitive or unusual cases may still require human review.

## Example Output

Input: "My order is late"

Output:
A polite and professional customer support email that:
- apologizes for the delay
- acknowledges the customer's concern
- requests the order number
- offers further assistance
